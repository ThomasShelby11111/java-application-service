## 📋 About java-application-service

This application follows the Java Application Service pattern to help manage and organize backend logic in Java applications. It uses clear structures to handle tasks, making apps easier to maintain and grow. The software focuses on handling operations in a clean way, separating responsibilities between different parts of the program.

The core ideas include organizing commands and processes using patterns such as mediator and orchestrator. These patterns direct how different parts of your application talk to each other without unnecessary dependencies. This approach improves clarity and reduces complexity.

Even if you are new to programming, you do not need to worry about how it works internally. This guide explains how to get the app running on Windows with simple steps.

## 🔍 How to Use java-application-service

This application runs on your Windows PC and uses a simple interface to manage different application services.

### Main Features:

- Organize commands and requests in your Java application.  
- Handle each task clearly through service layers.  
- Support for mediator pattern to reduce tight coupling.  
- Coordinate tasks using orchestrators for smooth processing.  
- Easy to maintain and extend as your app grows.

You do not need programming skills to start. The interface guides you through adding tasks and viewing their status.

### Opening a Service Task

- Use the menu to create a new service task.  
- Fill in the required fields about the task's role.  
- Click "Run" to execute the managed operation.

### Viewing Logs

- The app provides a log area to see ongoing process details.  
- This helps understand what happens during each service call.  

### Closing the Application

- Use the "Exit" button or close the window like any other program.
